# Documentación Técnica - NIDIA Flow

Este directorio contiene documentación técnica detallada del sistema.

## Documentos Disponibles

### [architecture.md](./architecture.md)
Documentación completa de la arquitectura del sistema, incluyendo:
- Diagramas de arquitectura de alto nivel
- Diagramas de componentes
- Diagramas de secuencia
- Modelos de datos
- Patrones de diseño
- Estrategias de escalabilidad

### [multi-tenant-architecture.md](./multi-tenant-architecture.md)
Documentación específica sobre la arquitectura multi-tenant:
- Modelo Database-per-Tenant
- Servicios clave
- Seguridad y control de acceso
- Provisión de bases de datos
- Optimizaciones de rendimiento

### [roles-and-permissions.md](./roles-and-permissions.md)
Documentación del sistema de roles y permisos:
- Jerarquía de roles
- Categorías de permisos
- Implementación con guards y decoradores
- Ejemplos de uso
- Mejores prácticas

## Diagramas

Todos los diagramas están creados con Mermaid y pueden visualizarse en:
- GitHub (renderizado automático)
- Editores que soporten Mermaid (VS Code con extensión)
- [Mermaid Live Editor](https://mermaid.live)

## Actualización de Documentación

Al realizar cambios significativos en la arquitectura, actualizar los diagramas correspondientes en los documentos.
